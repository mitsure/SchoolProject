# Step2 傷害カテゴリ別解析

## 1. 目的

Step1で確認した全体分布を基に、傷害カテゴリごとの解析対象データを作成し、学校種・学年・性別・発生場所・発生場面・通学方法・競技種目・年度の違いを比較する。

Step2は、次の2群の処理で構成する。

1. 各傷害カテゴリの事例を元データから抽出する
2. 全傷害カテゴリを同じ形式でクロス集計して比較する

## 2. ジャッカード係数

**Step2では使用しない。**

Step2は条件抽出と記述統計によるカテゴリ比較であり、特徴間または事例間の類似度評価ではないため。

ジャッカード係数は後続のJaccard解析Stepで使用する。

## 3. 入力

```text
DB/shougai(2025.01.31).csv
```

## 4. Pythonの配置

```text
file/Step2_傷害カテゴリ別解析/
```

## 5. 出力先

```text
CreateData/Step2_傷害カテゴリ別解析/
├─ カテゴリ別抽出データ/
└─ カテゴリ比較集計/
```

出力フォルダは各Pythonが自動生成する。同名CSVは上書きする。

## 6. カテゴリ別抽出データ

以下の12カテゴリを個別CSVとして抽出する。

- Step2-1：歯牙障害
- Step2-2：外貌・露出部分の醜状障害
- Step2-3：視力・眼球運動障害
- Step2-4：精神・神経障害
- Step2-5：手指切断・機能障害
- Step2-6：胸腹部臓器障害
- Step2-7：上肢切断・機能障害
- Step2-8：下肢切断・機能障害
- Step2-9：せき柱障害
- Step2-10：聴力障害
- Step2-11：足指切断・機能障害
- Step2-12：そしゃく機能障害

各CSVは元データの15列を維持する。

## 8. 実行方法

例：

```bash
python file/Step2_傷害カテゴリ別解析/Step2-1_傷害カテゴリ別解析_歯牙障害抽出.py
python file/Step2_傷害カテゴリ別解析/Step2-13_傷害カテゴリ別解析_学校種別比較.py
```

各Pythonは単独実行できる。

## 9. 実行後の確認

- 抽出件数がStep1-4の傷害種別件数と一致するか
- 抽出CSVの種別が対象カテゴリのみになっているか
- 元データの15列が維持されているか
- カテゴリ比較表の件数合計が元データ総件数と一致するか
- カテゴリ内割合がカテゴリごとに概ね100％になるか
- 欠損が異常に多い列がないか
- 表記揺れや想定外カテゴリがないか

## 10. Step2から分かること

- 傷害カテゴリごとの母数
- 各カテゴリに多い学校種・学年
- 性別分布
- 発生場所・発生場面の違い
- 通学方法・競技種目の違い
- 年度別分布

## 11. Step2では分からないこと

- 自由記述に含まれる事故原因
- 受傷までの流れ
- 特徴語
- 特徴間・事例間類似度
- テキスト追加前後の差

これらはStep3以降で解析する。
